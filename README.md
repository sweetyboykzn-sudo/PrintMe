# PrintMe — конструктор принтов (React + Node.js)

## Запуск
npm run install:all
npm run dev

- Сайт: http://localhost:5173
- Админка: http://localhost:5173/admin (пароль admin123)
- API: http://localhost:4000

## После запуска
1. Админка -> Основы и цвета
2. Загрузить мокапы front/back
3. Прописать BOT_TOKEN и ADMIN_CHAT_ID в .env
4. По желанию ключи ЮKassa
